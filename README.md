# LopesBot AI

Plataforma SaaS de Funcionários Virtuais com Inteligência Artificial.

## Visão

Criar uma plataforma capaz de automatizar atendimento,
vendas, agendamentos e suporte para pequenas empresas.

## Objetivo

Criar um funcionário virtual com IA integrado ao WhatsApp,
capaz de atender clientes 24 horas por dia.

## Tecnologias

### Frontend
- React
- TypeScript
- Vite

### Backend
- NestJS
- Node.js

### Banco de Dados
- PostgreSQL

### Infraestrutura
- Docker
- Linux
- Nginx

### Inteligência Artificial
- OpenAI API
- RAG
- Embeddings
